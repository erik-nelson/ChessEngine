# ChessEngine

Been playing some chess recently, so I decided to make an AI and GUI to improve
on my mistakes.

Only supported on Mac OSX.

To play:

```bash
git clone git@github.com:erik-nelson/AI_chess_engine.git
cd ChessEngine/src
./play
```
